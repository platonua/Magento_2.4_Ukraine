- [x] Провести оплату використовуючи тестові реквізити.

## Встановлення:

* Розпакуйте архів в корінь сайту.

* Встановіть модуль з адмін панелі Admin->System->Web Setup Wizard->Module Manager->Platon_PlatonPay->enable

* Включіть та налаштуйте метод оплати Admin->Stores->Configuration->SALES->Payment Methods->OTHER PAYMENT METHODS->Platon

* В налаштуваннях вказати ключ та пароль. Також можете змінити назву платіжного методу.

* Встановіть значення платіжного методу в статус Enabled.

* Якщо метод оплати не з'явився скиньте кеш System->Cache Manager
